# User Guide

This is a demo-oriented user guide for the current repository state.

## Basic Flow

1. Start the frontend.
2. Log in using the demo/offline mode or connect the backend if configured.
3. Navigate to handover, DSS, dashboards, history, and settings through the main UI.
4. Treat online synchronization as partial unless you have configured the backend explicitly.

## Important Note

The repository is an independent case study. Any operational process names or domain labels describe the technical scenario, not a live institutional deployment.
